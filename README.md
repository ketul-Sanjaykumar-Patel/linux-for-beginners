## 📖 What is this?

This repo is a **complete Linux learning kit** for absolute beginners.
It includes everything you need to go from zero to confident Linux user:
- A step-by-step written guide
- Real working scripts you can run
- Ready-to-use config files
- Quick reference cheatsheets
- Hands-on exercises with answers
- Embedded Linux tools (Raspberry Pi, GPIO)
